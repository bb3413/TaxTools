Tax Tools

This sections contains information that applies to all of the tax tools. The current set of tax tools are:

  • Age Calculator
  • Alternative Minimum Tax (AMT) Calculator
  • Enhanced Deduction for Seniors Calculator
  • Estimated Tax Calculator (federal)
  • Estimated Tax Calculator (California)
  • Required Minimum Distribution (RMD) Calculator
  • Simple Math Calculator
  • Small Business Tax Calculator
  • Social Security - Taxable Benefits Calculator
  • Social Security - Lump Sum Calculator
  • State Income Tax Refund Calculator

All of the tax tools use a common set of colors, and when it does not get in the way of the functionality, they have a similar look and feel. The input fields, where you enter data, are always green. And, the output fields, where the results are displayed, are always blue.

Tooltips

Many of the tools provide tooltips. These are small popup displays that show up when you move the mouse over one of the fields. They provide a short explanation of the purpose of the fields. When available, tooltips are provided for both input fields to help you enter the correct information, and for output fields to explain what the information means.

Numeric Input

For any input field where a numeric value is expected, dollar signs($) and commas(,) are ignored. This allows you to enter a number as, for example, "$40,000", "$40000", or "40000". In addition, numeric fields can contain simple arithmetic expressions. For example, you can enter "($40,000 + 4,000) * 0.075" to enter 7.5% of 44000. All numeric input will be rounded to the nearest whole number. The expression evaluator supports the following functionality:

  • Numbers with decimal points
  • Operators for addition (+), subtraction (-), multiplication (*), and division (/)
  • Unary minus operator (-)
  • Parentheses
  • Operator precedence for parentheses, unary minus, and for doing multiplication and division before addition and subtraction.

Debugging Information

Some of the tools can display additional information that shows internal calculations and intermediate results. Enter any of the following keywords in any input field to see this information. The keywords are not case-sensitive and can be entered in any order. Separate the input value and one or more keywords using commas or whitespace.

  • Debug - Display intermediate results for the current tool.
  • IncTax - Show the worksheet results for the federal income tax calculation.
  • SSTax - Show the worksheet results for the amount of Social Security that is taxable calculation.
  • SETax - Show the worksheet results for the self-employment tax calculation.
  • Trace - Logs function entry and exit to the console log.

Version Number

Each of the tax tools displays the version number, which is the same for all tax tools. The version number is always displayed as a link; if you click on it, it will take you to the version information log. There is only one version number and one version log for all the tax tools. This simplifies tracking changes since there is so much commonality and code sharing in the tools.

The version number consists of two or three numbers separated by dots (.). The first part is the most significant part and is set to the current tax year. It is incremented in the fall after the tax tables have been updated for the current tax year. The second part of the version number is initially set to 0 and is incremented each time a new version is released.

The third part of the version number is not present on released version of the tools. It is incremented when there is a change that can be seen by the user or that affect the behavior of one of the tools, or an internal change that may affect the reliability of the tools. Minor changes, such as cosmetic changes and changes to the wording in messages, may or may not be logged. When the third part of the version number is present, it is appended with an at sign(@), which indicates that that version is under development; it is not ready to be used.
